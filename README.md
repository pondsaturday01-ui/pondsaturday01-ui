# 🤖 Facebook Reels Auto-Bot (Ultimate Edition)

บอทโพสต์ Reels อัตโนมัติ รองรับหลายเพจ และมีหน้าจอควบคุม (Dashboard)

## 🚀 วิธีใช้งานด่วน
1. **เปิดหน้าจอควบคุม:**
   รันคำสั่ง: `streamlit run dashboard.py`
2. **กดปุ่ม Start Bot:** ที่หน้าเว็บที่เด้งขึ้นมา
3. **ดูรายงาน:** เลื่อนลงมาข้างล่างเพื่อดูว่าวันนี้โพสต์ไปกี่คลิปแล้ว

## 🔧 การตั้งค่า (config.yaml)
- **PAGE_MAPPINGS:** ใส่ลิสต์เพจ Facebook ที่ต้องการให้บอทไปโพสต์
- **schedule_times:** ใส่เวลาที่ต้องการให้โพสต์ (เช่น "08:00", "18:30")
- **video_folder:** ที่อยู่ของโฟลเดอร์คลิป

## 📁 ไฟล์สำคัญ
- `bot.py` = สมองสั่งการ (Loop & Schedule)
- `uploader.py` = แขนขา (กดปุ่ม Facebook)
- `bot_brain/` = โฟลเดอร์เก็บข้อมูล Login (ห้ามลบ! ไม่งั้นต้องล็อกอินใหม่)
